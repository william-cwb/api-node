# API NODE

## Utilizado:
- Sequelize ORM
- Postgres
- Express

Para executar o projeto:

Instale as dependências
```
$ yarn install 
```

Configure as conexoes do banco no arquivo: src/config/database.js

Faça o migrate
```
$ yarn sequelize db:migrate
```

e acesse os endpoint aqui, utilizando o Insomnia
